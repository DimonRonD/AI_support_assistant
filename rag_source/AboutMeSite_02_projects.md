# AboutMeSite — проекты и кейсы

## Проект 1. Nutribot

### Кратко
Nutribot — Telegram-бот для персональных рекомендаций по питанию и тренировкам.

### Задачи, которые решает
- первичный сбор данных о пользователе;
- автоматическая выдача базовых рекомендаций;
- поддержка вовлеченности клиентов в формате диалога.

### Польза для бизнеса
- снижение нагрузки на специалистов;
- масштабирование без пропорционального роста штата;
- ускорение запуска новых продуктовых сценариев.

## Проект 2. Шмавито

### Кратко
Шмавито — онлайн-платформа аренды и продажи вещей в формате маркетплейса.

### Задачи, которые решает
- быстрый запуск сервиса объявлений;
- организация каталога, поиска и оформления взаимодействий;
- поддержка роста аудитории продавцов и покупателей.

### Польза для бизнеса
- быстрее выход на рынок;
- ниже стартовые затраты по сравнению с разработкой с нуля;
- гибкая модель монетизации.

## Проект 3. Pusplexity

### Кратко
Pusplexity — AI-платформа для работы с изображениями и документами через Telegram и веб-интерфейс.

### Задачи, которые решает
- ускорение производства визуального контента;
- обработка изображений по текстовым запросам;
- вопросы-ответы по внутренним документам.

### Польза для бизнеса
- экономия времени на рутинных операциях;
- доступ к знаниям компании в удобном формате;
- ускорение внутренних и маркетинговых процессов.
