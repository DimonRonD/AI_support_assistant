# Инструкции по работе с чатом

Этот файл используется как источник знаний для ПетлиБота.

## Как очистить память?

Пользователь может очистить память чата двумя способами:

1. Нажать кнопку **«Очистить память»** в окне чата.
2. Отправить в чат команду: **«очистить память»**.

После очистки бот подтверждает действие и предлагает продолжить диалог с чистой историей.

## Дополнение для консультирования

- Память привязана к email пользователя.
- Бот учитывает последние 10 вопросов пользователя.
- При повторном входе бот может напомнить, на каком вопросе остановились в прошлой беседе.
