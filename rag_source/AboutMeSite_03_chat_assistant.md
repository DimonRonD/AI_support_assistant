# AboutMeSite — онлайн-консультант ПетлиБот

## Назначение

ПетлиБот — встроенный консультант сайта AboutMeSite.  
Он помогает посетителю быстро получить информацию о проектах, кейсах и релевантных услугах.

## Сценарий начала общения

1. Пользователь открывает чат.
2. Указывает имя и email.
3. После этого становится доступно окно диалога.

## Ограничения и правила общения

- Максимальная длина сообщения пользователя: 250 символов.
- Если длина превышена, показывается сообщение о превышении лимита до отправки.
- Пока формируется ответ, в чате отображается уведомление: «Подготавливаю ответ...».
- Бот отвечает вежливо, корректно и по теме сайта.
- Если вопрос не относится к сайту/проектам, бот сообщает, что консультирует только по проектам сайта.

## Память диалога

- Память привязана к email пользователя.
- Сохраняются последние 10 вопросов.
- При повторном входе бот может напомнить, на каком вопросе остановились.
- Память можно очистить:
  - кнопкой «Очистить память»;
  - командой «очистить память».

## Логи

- Ведутся логи вопросов и ответов.
- Ключ сессии: email + дата.
- Есть публичная страница логов с последовательностью диалога.
- Email в публичном выводе маскируется.

## Как бот формирует ответы

- Использует базу знаний, загруженную из `rag_source`.
- Для поиска контекста применяет ChromaDB.
- Финальный ответ формирует через OpenAI-модель, настроенную в конфиге проекта.
- При старте проекта выполняется автосинхронизация RAG-коллекции с файлами из `rag_source`.
