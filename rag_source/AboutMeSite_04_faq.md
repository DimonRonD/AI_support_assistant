# AboutMeSite FAQ — частые вопросы и ответы

## 1) Какие у вас есть проекты?

У нас представлены ключевые проекты:
- Nutribot (Telegram-бот для рекомендаций по питанию и тренировкам),
- Шмавито (онлайн-платформа в формате маркетплейса),
- Pusplexity (AI-платформа для работы с изображениями и документами через Telegram и Web),
- кейсы по корпоративным сайтам и AI-ассистентам поддержки.

## 2) Какие проекты работают с Telegram?

С Telegram работают несколько направлений:
- Nutribot,
- Pusplexity,
- AI-боты/ассистенты под задачи бизнеса.

При необходимости можно уточнить, какой сценарий Telegram нужен: продажи, поддержка, контент или внутренняя автоматизация.

## 3) Какие проекты подходят для фитнеса и тренировок?

В первую очередь Nutribot: он создан для рекомендаций по питанию и тренировкам, сбора данных пользователя и повышения вовлеченности через диалог.

## 4) Что делает Pusplexity?

Pusplexity помогает:
- обрабатывать и генерировать изображения;
- получать ответы по загруженным документам;
- ускорять внутренние процессы через AI-интерфейс в Telegram и Web.

## 5) Как выбрать проект под мой бизнес?

Ориентируемся на 3 вещи:
1. Цель (продажи, поддержка, автоматизация, контент).
2. Канал взаимодействия (Telegram, Web, оба).
3. Приоритет (быстрый MVP, масштабирование, снижение затрат).

После этого выбирается наиболее подходящий сценарий и этапы запуска.

## 6) Как очистить память чата?

Два варианта:
- нажать кнопку «Очистить память» в окне чата;
- отправить команду «очистить память».

## 7) Почему бот не отвечает на вопрос не по сайту?

ПетлиБот консультирует только по проектам и услугам этого сайта.  
Если вопрос внешний, бот попросит уточнить запрос в рамках проектов сайта.

## 8) Что запоминает чат?

Чат запоминает последние 10 вопросов пользователя по email, чтобы поддерживать контекст диалога.  
Эту память можно очистить по запросу.

## 9) Можно ли посмотреть историю диалогов?

Да, у проекта есть публичная страница логов с последовательностью вопросов и ответов.  
Email в публичном выводе маскируется.

## 10) Что нужно для старта проекта с вами?

Обычно достаточно:
- кратко описать задачу и желаемый результат;
- указать приоритеты по срокам и бюджету;
- обозначить каналы (Telegram, сайт, CRM и т.д.).

После этого можно предложить понятный пилотный формат запуска.
