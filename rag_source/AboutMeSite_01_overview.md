# AboutMeSite — общее описание проекта

## Что это за проект

AboutMeSite — персональный сайт Дмитрия Петлина для презентации digital- и AI-проектов, демонстрации кейсов и сбора клиентских заявок.  
Сайт ориентирован на потенциальных заказчиков, которым нужны:
- разработка Telegram-ботов;
- автоматизация бизнес-процессов;
- запуск онлайн-сервисов и веб-платформ;
- AI-интеграции для повышения эффективности.

## Основная цель сайта

1. Показать экспертизу через реальные кейсы.
2. Дать понятную навигацию по услугам и проектам.
3. Упростить первый контакт с потенциальным клиентом.
4. Предоставить онлайн-консультанта (ПетлиБот) для быстрых ответов по проектам сайта.

## Целевая аудитория

- Владельцы малого и среднего бизнеса.
- Руководители digital-направлений.
- Команды, которым нужна автоматизация рутины.
- Компании, планирующие запуск AI-продуктов или AI-функций.

## Ключевая ценность

Сайт помогает быстро понять:
- какие задачи уже решались;
- какие результаты это дало бизнесу;
- какой формат проекта можно запустить под задачу клиента.
